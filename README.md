## 1、前言
基于区块链技术的比特币从2009年的0.003美元涨到2017年12月18日 19013美元，上涨600多万倍，创造了一个新时代的财富传奇。

2010年，美国的程序员拉丝勒•豪涅茨(Laszlo Hanyecz)用1万枚比特币交换了两块价值25美金的披萨，相当于用了1.9亿美元买了一块披萨，成为“史上最贵披萨”。
